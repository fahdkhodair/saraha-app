Input validation and error handling

MongoDB database for storing users and messages securely

RESTful API endpoints for seamless frontend integration

Tech Stack:

Backend: Node.js, Express.js

Database: MongoDB (NoSQL)

Authentication: JWT

Tools: Postman for testing APIs

How to Run / Demo:

Clone the repository: git clone <repo-link>

Install dependencies: npm install

Configure .env file with MongoDB connection string and JWT secret

Start the server: npm start

Test endpoints using Postman
