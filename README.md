# Saraha App (Node.js + Express + MySQL)

A REST API for anonymous messaging app.

## Features
- User registration & login (JWT Authentication)
- Message CRUD (Create / Read / Update / Delete)

## Tech Stack
- Node.js
- Express.js
- MySQL
- JSON Web Tokens (JWT)

## How to Run
1. Clone the repo
2. Run `npm install`
3. Setup database
4. Run `npm start`
